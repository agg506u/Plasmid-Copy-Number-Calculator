# Plasmid Copy Number Calculator

A simple Google Colab tool to calculate the copy number of circular double-stranded DNA (plasmid) based on its specific sequence composition and concentration.

## Features
- **Sequence-Specific MW**: Calculates molecular weight based on exact A, T, G, and C counts.
- **Circular DNA Correction**: Considers dehydration synthesis (N times condensation for circular forms).
- **Double-Stranded Output**: Calculates correct dsDNA molecular weight and copies/µL.

## How to Use
1. Click the **Open in Colab** badge above.
2. Run the code cell in Google Colab.
3. Paste your 5'→3' sequence and enter the concentration (ng/µL).
4. Click **計算 (Calculate)** to get the results.
